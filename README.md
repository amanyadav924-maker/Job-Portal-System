# Job Portal System (Web-based)
Overview:
A simple Job Portal web application built with Core Java, JDBC, Servlets, JSP and deployable on Apache Tomcat.
Roles: ADMIN, EMPLOYER, JOBSEEKER.
Prerequisites:
- Java 8+
- Maven (optional if using IDE)
- MySQL / MariaDB
- Apache Tomcat 9+
- IDE: IntelliJ / Eclipse / NetBeans
Setup Summary:
1. Create MySQL database: CREATE DATABASE job_portal;
2. Run sql/schema.sql to create tables.
3. Copy src/main/resources/db.properties.sample to src/main/resources/db.properties and fill DB credentials.
4. Build with Maven or import into IDE.
5. Deploy WAR to Tomcat or run from IDE
